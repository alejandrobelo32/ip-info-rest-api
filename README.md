# API REST Ip Info

Api para obtener informacion de una IP.
Construida con Slim Framework (PHP) y MySQL

## Install the Application

Intalar un servidor del tipo xampp (apache, php, MySql)
Levantar la BD incluida en el proyecto en el servidor SQL
Copiar el proyecto en la raiz del servidor web (en el caso de xampp es htdocs)

## EndPoints 

http://localhost/slim_app/public/ip/get-info/{ip}
